
		ЗАДАНИЕ:
BEEGEEK наконец открыл свой банк в котором используются специальные банкоматы с необычным паролем.
Действительный пароль BEEGEEK банка имеет вид a:b:c, где a, b и c – натуральные числа. 
Поскольку основатель BEEGEEK фанатеет от математики, то он решил:

число a – должно быть палиндромом;
число b – должно быть простым;
число c – должно быть четным.
Напишите функцию is_valid_password(password), которая принимает в качестве аргумента строковое значение 
пароля password и возвращает значение True если пароль является действительным паролем BEEGEEK банка и False в противном случае.

		РЕШЕНИЕ:
def is_valid_password(password):
   x = [i for i in password.split(':')]
   if len(x) == 3:
      sp = x[0]
      sp_2 = x[1]
      sp_3 = x[2]
      if sp[::] == sp[::-1]:
         y = int(sp_2)
         if len([i for i in range(1, y + 1) if y % i == 0]) == 2:
            j = int(sp_3)
            if  j % 2 == 0:
               return True
            else:
               return False
         else:
            return False
      else:
         return False
   else:
      return False
print(is_valid_password(input()))
