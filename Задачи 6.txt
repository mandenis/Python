	ЗАДАЧА
	Получить число из строки
ОПИСАНИЕ:
Напишите функцию, которая удаляет из строки все нецифровые символы и преобразует оставшиеся в числа. Например: "hell5o world6ld" -> 56

Функция:
getNumberFromString(s

	РЕШЕНИЕ
def get_number_from_string(string):
    return int(''.join(x for x in string if x.isdigit()))

ОПИСАНИЕ:
Создайте функцию, которая принимает строку и один символ и возвращает целое число, соответствующее количеству вхождений второго

Если вхождений не найдено, должно быть возвращено число 0.

("Hello", "o")  ==>  1
("Hello", "l")  ==>  2
("", "z")       ==>  0
str_count("Hello", 'o'); // returns 1
str_count("Hello", 'l'); // returns 2
str_count("", 'z'); // returns 0

	РЕШЕНИЕ
def str_count(strng, letter):
    return strng.count(letter)
